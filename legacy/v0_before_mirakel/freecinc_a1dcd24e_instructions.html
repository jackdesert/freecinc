your TaskWarrior client.

Your keys have been generated...let's get started!

Step 1:

Prereqs

Make sure TaskWarrior 2.3.0 or greater is installed
task --version
Step 2:

Save Keys

Save the following files to your ~/.task directory
Step 3:

Configuration

Run the following in a terminal (one block at a time because you have to confirm each step)
task config taskd.server freecinc.com:53589

task config taskd.key ~/.task/freecinc_a1dcd24e.key.pem

task config taskd.certificate ~/.task/freecinc_a1dcd24e.cert.pem

task config taskd.ca ~/.task/freecinc_a1dcd24e.ca.pem

task config taskd.credentials \
'FreeCinc/freecinc_a1dcd24e/c90446f1-5d74-4b5e-abdc-12da6a4f06bd'
Step 4:

Verify

Run the diagnostic tool
task diagnostics
and make sure that in the output CA, Cert, and Key are all "readable". Also ensure that you have a value for Server and a value for Creds.
Step 5:

Sync

The moment you've been waiting for!
task sync init
Step 6:

Save This

(optional) If you want to set up another computer to sync with the same account, click "File > Save As" from your browser and save this page. Open the resulting html file on your other computer, and complete steps 1-5.
